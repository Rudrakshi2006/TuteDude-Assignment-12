# Selenium Automation Assignment

## Objective

In this assignment I learned how to use Selenium with Python. I practiced opening websites, searching in Google, clicking links, refreshing pages and getting data from a webpage.

## Software Used

* Python
* Selenium
* Google Chrome
* VS Code

## Files

* automation.py

## How to Run

1. Install selenium

```
pip install selenium
```

2. Run the program

```
python automation.py
```

## What my program does

* Opens Google
* Searches "Python Selenium"
* Opens Books to Scrape website
* Refreshes the page
* Opens Travel category
* Prints first 5 book names
* Prints the first book price
* Closes the browser

## Concepts Used

* webdriver.Chrome()
* driver.get()
* maximize_window()
* By.NAME
* By.CLASS_NAME
* By.LINK_TEXT
* By.XPATH
* send_keys()
* click()
* refresh()
* find_elements()
* print()
* quit()

## What I Learned

From this assignment I learned how Selenium controls a browser. I also learned different ways to find elements on a webpage and how to extract text from them.
